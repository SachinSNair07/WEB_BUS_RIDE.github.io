About the Game

Born on Kerala's Roads

Kerala Bus Mini Simulator is a browser-based arcade game inspired by the chaotic, vibrant, and legendary bus culture of Kerala — where every journey is an adventure.

This is a prototype for a much larger project — TVM Bus Simulator — a full-fledged simulation set in Thiruvananthapuram with authentic routes, real landmarks, and deep passenger mechanics.

Built entirely in HTML5 Canvas and JavaScript, this mini version captures the essence of dodging traffic at breakneck speed while keeping your passengers safe.

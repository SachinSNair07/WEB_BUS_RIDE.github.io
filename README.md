# 🚌 Bus Mini Sim

A browser-based driving game inspired by real Kerala road conditions and bus driving experience.

---

## 🎮 Gameplay

- Control the bus using **Left / Right arrow keys**
- Avoid incoming traffic
- Survive as long as possible
- Score increases over time
- Difficulty increases gradually

---

## 🚀 Features

- Real-time game loop
- Traffic spawning system
- Collision detection
- Score tracking system
- Restart functionality
- Simple and responsive controls

---

## 🧠 Purpose

This project is a **mini prototype** of a larger game:

👉 TVM Bus Simulator  

It demonstrates core game development concepts such as:
- Vehicle movement
- Traffic AI basics
- Game loop design
